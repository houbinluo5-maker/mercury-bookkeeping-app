# Roadmap

This roadmap outlines planned improvements for Mercury Bookkeeping App, an open-source bookkeeping app starter for small ecommerce businesses.

## v0.1.0 - Initial Open-Source MVP

* Manual transaction tracking
* Revenue, COGS, expense, owner draw, and transfer categories
* Dashboard with bookkeeping metrics
* Mercury-style CSV import
* Duplicate transaction detection
* Receipt tracking
* Monthly close workflow
* Audit trail
* Monthly, quarterly, and annual summary exports
* Excel-compatible exports

## v0.2.0 - Setup and Documentation Improvements

* Improve local setup documentation
* Add more screenshots to the README
* Add sample Mercury-style CSV files
* Add example bookkeeping workflows for ecommerce sellers
* Improve Supabase setup guide
* Add troubleshooting guide for local development

## v0.3.0 - CSV Import Improvements

* Improve column matching for Mercury-style CSV files
* Add better import preview warnings
* Improve duplicate detection rules
* Add clearer category suggestions
* Add import summary reports
* Add more sample CSV test cases

## v0.4.0 - Reporting Improvements

* Improve monthly report layout
* Add better tax summary grouping
* Add CPA export package examples
* Add receipt package index improvements
* Add owner draw and member distribution summaries

## v0.5.0 - Reliability and Testing

* Add automated tests for CSV parsing
* Add tests for accounting classification rules
* Add tests for monthly close logic
* Add GitHub Actions CI workflow
* Improve TypeScript type safety
* Improve error handling for data import and export

## v1.0.0 - Production-Ready Starter

* Harden Supabase persistence
* Add stronger role-based access controls
* Improve audit log retention controls
* Improve backup and restore workflows
* Add better documentation for secure private deployment
* Prepare the project as a stable open-source starter for ecommerce bookkeeping

## Long-Term Ideas

* Optional integrations with ecommerce platforms
* Optional bank CSV format templates
* Multi-company support
* More advanced reconciliation workflows
* More flexible chart of accounts
* Better international currency support
* Improved mobile layout
