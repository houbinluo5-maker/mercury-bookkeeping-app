# Page Layouts

## Sidebar
Groups: Command, Imports, Finance Ops, Reports, System. The top brand area shows Mercury Bookkeeping, company name, tax year, and entity context.

## Dashboard
Executive finance cockpit:
- KPI row: revenue, net income, expenses, month close readiness.
- Finance health panel: missing receipts, needs review, unreconciled, possible duplicates.
- Quick actions: add transaction, import Mercury CSV, upload receipt, open reconciliation.
- CPA readiness card.
- Recent transactions.

## Transactions
Ledger workspace:
- Page header with export/add actions.
- Filter bar.
- Data table shell.
- Clear status badges and edit actions.

## Receipts
Document control workspace:
- Summary panels.
- Missing receipts queue.
- Receipt table shell.
- Upload controls remain visible and unchanged.

## Reconciliation Center
Finance operations command center:
- Readiness score.
- Issue summary.
- Top issues to fix first.
- Export panel.
- Issue sections by severity/workflow.

## Tax Package
CPA deliverable:
- Date/tax-year filter panel.
- Professional disclaimer.
- Closing status panel.
- Summary grouped by financial theme.
- Primary workbook export and grouped CSV exports.

## Settings
Structured system console:
- Company profile.
- Accounting settings.
- Supabase status.
- Backup/data tools.
- System maintenance with dangerous actions separated.
